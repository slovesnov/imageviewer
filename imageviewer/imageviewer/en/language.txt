Drag &amp; drop file to view directory which includes dropped file, starts from dropped file if file is supported.
If dropped file isn't supported then viewer starts from 1st supported image. One can drop directory to view it.
<i>Shortcuts list.</i>
<b>arrow right</b> or <b>KP_6</b> / <b>arrow left</b> or <b>KP_4</b> {
	LIST MODE scroll one row
	FIT/NORMAL MODES next/previous image in directory
}
<b>Page_Down</b> or <b>KP_3</b> / <b>Page_Up</b> or <b>KP_9</b> {
	LIST MODE scroll screen
	FIT/NORMAL MODES +/-sqrt (number of images in directory)
}
<b>Home</b> or <b>KP_7</b> / <b>End</b> or <b>KP_1</b> go to first/last image in directory
<b>mouse right button</b> or <b>O</b> { 
	open file/folder, use <i>select</i> button in dialog to open file or folder, <i>open</i> button works only with files
}
<b>vertical mouse scroll</b> {
	LIST MODE scroll one row
	FIT MODE ignored
	NORMAL MODE scroll image up/down if image higher than window
}
<b>horizontal mouse scroll</b> {
	LIST, FIT MODES ignored
	NORMAL MODE scroll image left/right if image wider than window
}
<b>F11</b> or <b>F</b> or <b>Escape</b> switch full screen mode
<b>+</b>, <b>-</b> for LIST MODE increase/decrease icon size otherwise switch to NORMAL MODE/FIT MODE
<b>L</b> - switch to LIST MODE
<b>E</b>, <b>R</b>, <b>T</b> - rotate image by 90&#176;, 180&#176;, 270&#176;
<b>H</b>, <b>V</b> - flip image horizontally or vertically
<b>Delete</b> or <b>KP_Decimal</b> {
	NORMAL, FIT MODES - remove current image with confirm dialog and goes to next image in directory
	LIST MODE - change ascending/descending list order
}
<b>F1</b> - show help
