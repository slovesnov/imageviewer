(GDK_KEY_Right, GDK_KEY_KP_6) / (GDK_KEY_Left, GDK_KEY_KP_4) {
	Режим LIST прокручивает одну строку
	FIT/NORMAL MODES следующее/предыдущее изображение в DIRECTORY
}
(GDK_KEY_Page_Down, GDK_KEY_KP_3) / (GDK_KEY_Page_Up, GDK_KEY_KP_9) {
	Экран прокрутки в режиме СПИСОК
	FIT/NORMAL MODES +/-sqrt (количество изображений в DIRECTORY)
}
(GDK_KEY_Home, GDK_KEY_KP_7) / (GDK_KEY_End, GDK_KEY_KP_1) переход к первому/последнему изображению в ДИРЕКТОРИИ
mouse_right, O { 
	открыть файл/папку для выбора папки или файла. Используйте кнопку "выбрать" в диалоге для открытия файла или папки, кнопка "открыть" работает только с файлами
}
вертикальная прокрутка {
	Режим LIST прокручивает одну строку
	режим прокрутки игнорируется
	НОРМАЛЬНЫЕ РЕЖИМЫ прокрутка изображения вверх/вниз, если изображение выше окна
}
горизонтальная прокрутка мышью {
	игнорируются режимы LIST, FIT
	НОРМАЛЬНЫЕ РЕЖИМЫ прокрутки изображения влево/вправо, если изображение шире окна
}
GDK_KEY_F11, F, GDK_KEY_Escape переключение в полноэкранный режим
+/- для LIST_MODE увеличение/уменьшение размера иконки, иначе переключение в НОРМАЛЬНЫЙ РЕЖИМ/ФИТОГРАФИЧЕСКИЙ РЕЖИМ
l - переключиться в режим СПИСОК
e, r, t - повернуть изображение на 90, 180, 270 градусов против часовой стрелки
GDK_KEY_Delete, GDK_KEY_KP_Decimal {
	NORMAL, FIT MODES - удалить текущее изображение с диалогом подтверждения и переходом к следующему изображению в ДИРЕКТОРИИ
	РЕЖИМ СПИСКА - изменение по возрастанию/убыванию
}
