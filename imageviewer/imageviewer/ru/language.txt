Перетащите файл для просмотра каталога, который включает файл. Если файл не поддерживается - просмотр начнётся 
с первого поддерживаемого изображения. Можно также перетаскивать каталоги.
<i>Список горячих клавиш.</i>
<b>стрелка вправо</b> или <b>KP_6</b> / <b>стрелка влево</b> или <b>KP_4</b> {
	Режим LIST прокручивает одну строку
	FIT/NORMAL MODES следующее/предыдущее изображение в каталоге
}
<b>Page_Down</b> или <b>KP_3</b> / <b>Page_Up</b> или <b>KP_9</b> {
	Экран прокрутки в режиме СПИСОК
	FIT/NORMAL MODES +/-sqrt (количество изображений в каталоге)
}
<b>Home</b> или <b>KP_7</b> / <b>End</b> или <b>KP_1</b> переход к первому/последнему изображению в каталоге
<b>правая клавиша мыши</b> или <b>O</b> { 
	открыть файл/папку, используйте кнопку <i>выбрать</i> чтобы открыть файл или папку, кнопка <i>открыть</i> работает только с файлами
}
<b>вертикальная прокрутка мыши</b> {
	Режим LIST прокручивает одну строку
	режим прокрутки игнорируется
	НОРМАЛЬНЫЙ РЕЖИМ прокрутка изображения вверх/вниз, если изображение выше окна
}
<b>горизонтальная прокрутка мыши</b> {
	игнорируются режимы LIST, FIT
	НОРМАЛЬНЫЙ РЕЖИМ прокрутки изображения влево/вправо, если изображение шире окна
}
<b>F11</b> или <b>F</b> или <b>Escape</b> переключение в полноэкранный режим
<b>+</b>, <b>-</b> для режима списка увеличение/уменьшение размера иконки, иначе переключение в НОРМАЛЬНЫЙ РЕЖИМ/ФИТОГРАФИЧЕСКИЙ РЕЖИМ
<b>L</b> - переключиться в режим списка
<b>E</b>, <b>R</b>, <b>T</b> - повернуть изображение на 90, 180, 270 градусов
<b>Delete</b> или <b>KP_Decimal</b> {
	NORMAL, FIT MODES - удалить текущее изображение с диалогом подтверждения и переходом к следующему изображению в каталоге
	РЕЖИМ СПИСКА - изменение по возрастанию/убыванию
}
<b>H</b> или <b>F1</b> - показать помощь
